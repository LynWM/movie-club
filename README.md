# The Seventh Art Society Website

This is a simple website for The Seventh Art Society Movie Club. 

Allows users to view current selection, meeting date and search for movies. 

## It allows users to:

    1. Search for movies from the OMDb API
    2. Add movies to watchlist.(stored in json)
    3. Add ratings and comments to movies in watchlist.
    4. Remove movies from watchlist

### Main Objectives

- The purpose of this project was to demonstrate the use of fetching resources from external APIs and utilizing other HTTP verbs using the JSON server.

- These verbs are: GET, POST, UPDATE, DELETE, PUT , PATCH.
